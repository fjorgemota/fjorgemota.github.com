Contato

Olá!

Bom saber que você quer me contatar. :)

Para mandar um e-mail, use o formulário de contato abaixo, terei o maior prazer em lhe responder. :)

Note algumas coisas:

  • Todos os campos são obrigatórios
  • Sua mensagem será respondida em até 3 dias.
  • Se você quiser fazer um orçamento, use o formulário para orçamentos, que possui mais perguntas para algo do tipo.
  • Se você quiser pedir ajuda em um código seu, use o formulário para orçamentos também, e prepare-se para pagar. (afinal nada é de graça neste mundo, né?)
  • Se você encontrar alguma falha em algum post meu, relate diretamente nos comentários do post.
  • Se o formulário de contato abaixo não estiver disponível, mande um e-mail diretamente para o meu e-mail com o assunto "Contato". Clique aqui e veja o meu e-mail.
