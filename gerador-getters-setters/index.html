Gerador de Getters e Setters

Coloque o trecho da definição de atributos da classe no campo abaixo e clique em "Gerar" para gerarmos Getters e Setters personalizados para você.

O gerador suporta as seguintes linguagens:

  • PHP
  • Javascript
  • Java
  • Python

E ele é capaz de detectar a linguagem sozinho. Vale lembrar que você não deve colocar quaisquer tags de início de definição da classe, mas apenas dos atributos. Segue alguns exemplos abaixo:

PHP: 

$nome;
$idade;

JavaScript:

this.nome;
this.idade;

Java:

String nome;
int idade;

Python:

nome
idade

Teste, e divirta-se. ;)

Observação: O gerador adiciona dois métodos adicionais, toArray e setRow, para permitir o fácil transporte de dados para o banco de dados ou para outra camada de transporte (e.g: JSON), se quiser, sinta-se a vontade para removê-las. ;)
